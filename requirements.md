# 全自動投資システム — 要件定義書

## プロジェクト概要
世界情勢・経済指標・テクニカル指標を収集・分析し、
仮想通貨を全自動で売買するシステム。
現在は元手ゼロのためペーパートレード（シミュレーション）で学習。
資金ができ次第、本番スイッチを入れる。

## フェーズ

### Phase 1（今）: ペーパートレード + 学習
- 情報収集
- AI分析
- 架空資金で売買シミュレーション
- 取引ログ蓄積

### Phase 2（資金調達後）: 本番運用
- 取引所APIと接続
- 実資金で自動売買
- 継続学習

## システム構成

```
collector.py   → ニュース・価格・指標の収集
analyzer.py    → Claude APIで市場分析・売買判断
trader.py      → ペーパートレードエンジン
logger.py      → 取引ログ・学習データ管理
main.py        → メインループ（定期実行）
```

## 収集する情報
- 仮想通貨価格（BTC/ETH/他）: CoinGecko API
- 世界ニュース: NewsAPI
- テクニカル指標: RSI・MACD・ボリンジャーバンド
- Fear & Greed Index

## 技術スタック
- Python 3.11+
- Claude API (claude-sonnet-4-6)
- ccxt（取引所連携、Phase 2）
- pandas・ta（テクニカル分析）
- SQLite（ログ保存）

## 売買ルール（初期）
- Claude APIが「買い」と判断 → 仮想資金の10%を購入
- Claude APIが「売り」と判断 → 保有資産を売却
- リスク管理: 1回の損失は仮想資金の5%まで
